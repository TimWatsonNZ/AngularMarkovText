Markov Text Generator:
This program generates text by analying how likely a given word follows another or how likely a given word follows a tuple of words.
To use, feed in some input and click Analyze, then click Generate.

Order controls the number of tuples to analyse, increasing the order should make the generated text more realistic but less random. Output length is simply how long the generated text will be.
